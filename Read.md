# 🚨 Emergency Service 

## 📌 Project Overview

The **Emergency Service** is a web-based application designed to provide quick access to essential emergency services. Users can easily find and contact important service numbers such as fire service, police, and other emergency contacts.

This project is currently a **basic frontend structure**, focused on core UI and essential interactions. It includes features like emergency calling, coin-based call system, copy functionality, and call history tracking. 

The coin system allows users to make calls by deducting coins from their balance, while each call is recorded in the history section for future reference. Users can also copy service numbers instantly for quick use.

This project is under continuous development, and more advanced features and improvements will be added in the future.
